# ⚡️ Introduction

Welcome to Recipe Book, an application that empowers users to create, search, and delete their favorite recipes with ease.

# 🔥 Features

- 🔖 **Add recipe**
- ⚛ **Delete recipe**
- 🔍 **Search recipe**

# ⚙️ Behind the scene

- ✨ **React motion component** (warning message portrait mode)
- 🔀 **Routing**
- 📂 **Manage JSON**
- 🎞️ **Multiple CSS Animations**
